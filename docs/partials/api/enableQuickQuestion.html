enableQuickQuestion

Description

Add this attribute to make an element (use a div) containing 'did you know?' comment. The answer will be shown after clicking anywhere in the box

  • e-question: The question being asked
  • e-answer: The answer to the question 
    <div enableQuickQuestion e-question="Did you know?" e-answer="No I didn't!"></div>

Usage

as attribute
<ANY enable-quick-question>
   ...
</ANY>
