enableSectionHeader

Description

Directive that places a section header with a background picture, it alt text and the section title

Usage

as element:
<enable-section-header
       picpath="{string}"
       picalt="{string}"
       title="{string}">
</enable-section-header>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
picpathstring

The path of the background image for the section header.

picaltstring

The alternative text for the header image, to be read by screenreaders.

titlestring

The title of the section
