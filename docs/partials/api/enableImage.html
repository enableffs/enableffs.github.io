enableImage

Description

Directive that creates an audio player with the audio id provided. The player expects and will provide sounds tracks in 2 formats, m4a and ogg.

Usage

as element:
<enable-image
       picname="{string}"
       picalt="{string}">
</enable-image>
as attribute
<ANY enable-image
     picname="{string}"
     picalt="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
picnamestring

The name of the picture to insert (relative to the media folder where the section is loaded from)

picaltstring

The alternative text for the picture to be read by screenreaders
