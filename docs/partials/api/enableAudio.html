enableAudio

Description

Directive that creates an audio player with the audio id provided. The player expects and will provide sounds tracks in 2 formats, m4a and ogg.

Usage

as element:
<enable-audio
       sndid="{string}">
</enable-audio>
as attribute
<ANY enable-audio
     sndid="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
sndidstring

The id of the local video file. This id will be used for the poster image and the subtitles
