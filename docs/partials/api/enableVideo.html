enableVideo

Description

Directive that creates a local video player based on the localmode paramter with the video id provided and the language for the subtitles.

Usage

as element:
<enable-video
       vididyt="{string}"
       vididlc="{string}"
       cclang="{string}"
       localmode="{boolean}">
</enable-video>
as attribute
<ANY enable-video
     vididyt="{string}"
     vididlc="{string}"
     cclang="{string}"
     localmode="{boolean}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
vididytstring

The id of the youtube video. This will bring up the youtube video player , as well as the CC attached to that video

vididlcstring

The id of the local video file. This id will be used for the poster image and the subtitles

cclangstring

The current language code. This will load the subtitles in the current portal language.

localmodeboolean

The mode that needs to be inserted in the page, true => local videos, false => youtube videos
