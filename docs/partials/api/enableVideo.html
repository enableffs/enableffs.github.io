enableVideo

Description

Directive that creates a local video player with the video id provided and the language for the subtitles.

Usage

as element:
<enable-video
       vidid="{string}"
       cclang="{string}">
</enable-video>
as attribute
<ANY enable-video
     vidid="{string}"
     cclang="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
vididstring

The id of the local video file. This id will be used for the poster image and the subtitles

cclangstring

The current language code. This will load the subtitles in the current portal language.
